# iris-species-classification
End-to-End Data Science project using Python, EDA with IQR method, and Random Forest Classifier
# Iris Species Classification Project

โปรเจกต์วิเคราะห์และจำแนกสายพันธุ์ดอกไอริส (Iris Dataset) ซึ่งเป็นส่วนหนึ่งของวิชาเรียนด้าน Data Science

## 📋 Project Overview
โปรเจกต์นี้ครอบคลุมกระบวนการ Data Science ตั้งแต่การสำรวจข้อมูลไปจนถึงการสร้างโมเดลเพื่อทำนายผล

## 🛠️ Key Features & Workflow
- **Exploratory Data Analysis (EDA):** วิเคราะห์ความสัมพันธ์ของข้อมูลผ่านกราฟอย่างน้อย 3 รูปแบบ
- **Data Cleaning & Quality:**
    - ตรวจสอบ Missing Values
    - ใช้ **IQR Method** ในการตรวจสอบ Outliers เพื่อรักษาคุณภาพของข้อมูล
- **Data Manipulation:** การทำ Group by และ Pivot table เพื่อเตรียมข้อมูล
- **Machine Learning:** พัฒนาโมเดล **Random Forest Classifier** โดยแบ่งข้อมูลแบบ Train/Test (80:20)

## 📊 Results
โมเดลสามารถทำนายผลได้อย่างแม่นยำ โดยวัดผลผ่านค่า Accuracy, Precision, Recall และ F1-score
